## 📘 Student Productivity Tracker (CLI - Python)

### 📌 Overview
A simple command line-based Python program designed to help students keep track of their study time and maintain a productive week.

This project aims to resolve a specific problem or need by utilizing fundamental programming concepts in a concise and efficient manner.

### ❗ Problem
A problem faced by most students, including:
- Keeping  track of actual study time
- Being consistent with study goals
- Understanding how productive they are

### ✅ Solution
A program that allows a user to:
- Write study hour
- View total productivity
- View study histories
- Set and maintain weekly target
- Reset all informations as and when required

### 🚀 Feature
- Log time a student has studied
- View his/her study time
- View his/her study history
- Set  goals
- Track progress 
- Reset all information
- Persistent storage of information in a file

### 🛠️ Tech Stack
- Python Programming Language
  - Core Programming Concepts Only
  - File Handling
  - Functions
  - Loops
  - Exception Handling
